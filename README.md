# Order Summary

The order summary component is challenge of [frontendmentor.io](https://frontendmentor.io) made in ReScript

## Getting started

#### Install dependencies

```shell
yarn
```

or

```shell
npm install
```


#### Build project
```shell
yarn res:build
```

or

```shell
npm run res:build
```


#### Run ReScript in Watch mode
```shell
yarn res:watch
```

or

```shell
npm run res:watch
```
